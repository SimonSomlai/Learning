CSS Units

Understanding the difference between em, rem, px, vh & vw & %.

EM/REM
Relative to (a) parent element. Translated into px;
  • REM (relative to the HTML element, doesn't inherit)
  • EM (relative to the nearest parent element, inherits)

REM: I'm text relative to the HTML DOCUMENT!

EM: I'm text relative to my inherited value!

PX
Standard pixel value

I'm just kick-ass clear pixel values!

VH/VW
Defined according to the viewport height and viewport width

VH: I'm text relative to the VH!

VW: I'm text relative to the VW!

Percentages
Defined according to the width of it's parent element

My box-model is relative to the width of my container but not my font-size!
